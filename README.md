Implemented 2D CFAR (Constant False Alarm Rate) algorithm for adaptive thresholding and target detection in radar signal processing using MATLAB.
# FMCW Radar Target Detection using CFAR

## 📡 Overview
This project implements an FMCW radar system for target detection using MATLAB. It includes range estimation, Doppler processing, and CFAR-based adaptive thresholding.

## 🚀 Features
- FMCW waveform generation
- Range detection using FFT
- Range-Doppler Map (2D FFT)
- CFAR detection algorithm
- Noise reduction and target extraction

## 🛠 Tools Used
- MATLAB
- Signal Processing Techniques (FFT, CFAR)

## 📊 Output
- Range FFT Plot
- Range-Doppler Map
- CFAR Detection Map

## ▶️ How to Run
1. Open MATLAB
2. Run `main.m`
3. Outputs will be displayed in figures

## 🎯 Applications
- Radar Systems
- Defense Applications
- Autonomous Vehicles
